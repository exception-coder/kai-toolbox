# ✅where条件的顺序影响使用索引吗？

> 题号：0946 ｜ 分类：05_MySQL

---

## 速记

- 假如有一个联合索引，（a,b），那么如下两个SQL的性能有差别么？

```java
SELECT * FROM my_table WHERE a = 'value' AND b = 'value2';
SELECT * FROM my_table WHERE b = 'value2' AND a = 'value';
```

- 因为有查询优化器的存在，字段的先后顺序并不重要，不信的话可以explain看一下上面两个SQL的执行计划，都是可以命中(a,b)的联合索引的。
