# ✅POI的如何做大文件的写入

> 题号：0524 ｜ 分类：21_Excel与文件处理

---

## 速记

- 大文件写入选 `SXSSFWorkbook`（流式版），不要用 `XSSFWorkbook`
- 原因：SXSSF 把行 flush 到临时磁盘文件，XSSF 全在内存

### 实测对比（1万行 × 100列 UUID）

| 方式 | 堆内存 |
|---|---|
| XSSFWorkbook | 1200+ MB |
| SXSSFWorkbook | ~148 MB |

- 设 `-Xmx150m`：XSSF 直接 OOM，SXSSF 正常完成
- API 用法跟 XSSF 完全一致，只需替换 `new XSSFWorkbook()` → `new SXSSFWorkbook()`
- 注意：SXSSF 不支持读、不支持随机访问已 flush 的行
