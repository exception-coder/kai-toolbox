# ✅为啥 like %xx不走索引？like xx%xxx走索引吗？为啥？

> 题号：0789 ｜ 分类：05_MySQL

---

## 速记

- 当使用 `LIKE '%xx'` 进行查询时，通配符 `%` 放在了字符串的前面。
- 而  `LIKE 'xx%'`时，`%` 通配符放在了字符串的后面。
- 使用 `LIKE 'xx%xxx'`时，和`LIKE 'xx%'`是一样的，他都可以用到左边的`xx`来进行索引查询。
- 这就像你要去图书馆查一本书，图书馆的书都是按照名称排序的，你知道你要找的书的第一个字，那么你就可以到和第一个字一样的书架上去直接查找。
