# ✅什么是POI，为什么它会导致内存溢出？

> 题号：0535 ｜ 分类：21_Excel与文件处理

---

## 速记

- POI = Apache 开源的 Office 文档处理工具
- xlsx 本质是 zip 压缩包，内部是多个 XML 文件（`xl/`、`docProps/`、`_rels/`）
- 一个 2MB 的 xlsx 解压后可能十几 MB → 内存实际占用远大于文件本身

### OOM 根因

- `XSSFWorkbook(InputStream)` → `OPCPackage.open(in)` → `new ZipPackage(in, READ_WRITE)`
- 该方法注释明确说：**把整个压缩包加载到内存**
- 大文件 → 解压后 XML + DOM 对象树同时驻留 → OOM

### 三种 Workbook

| 类 | 格式 | 内存特性 |
|---|---|---|
| `HSSFWorkbook` | .xls (97-2003) | 全内存 |
| `XSSFWorkbook` | .xlsx | 全内存，大文件易 OOM |
| `SXSSFWorkbook` | .xlsx 流式 | 部分行刷盘，仅写入用 |
