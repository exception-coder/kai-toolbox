# ✅XXL-JOB相比Spring中的@Scheduled有啥优势？

> 题号：0029 ｜ 分类：20_任务调度

---

## 速记

类比：`@Scheduled` ≈ synchronized，XXL-JOB ≈ 分布式锁。

| 维度 | @Scheduled | XXL-JOB |
|---|---|---|
| 分布式 | 单机；集群每台都跑 → 重复执行 | 调度中心加锁，集群只执行一次 |
| 任务持久化 | 纯 JVM 内存，重启丢调度状态 | 有自己的 DB，重启不影响调度 |
| 故障转移 | 单机挂任务挂 | 集群级，单点挂不影响 |
| 可视化 | 无 | Web 控制台：配置/记录/报错 |
| 动态修改 | 改代码 + 重启 | 控制台改完即生效 |
| 分片 | 不支持 | 支持广播分片，大任务拆给多台并行 |
| 其他 | 无 | 权限/日志/任务依赖 |
