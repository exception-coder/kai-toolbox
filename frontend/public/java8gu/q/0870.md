# ✅happens-before和as-if-serial有啥区别和联系？

> 题号：0870 ｜ 分类：03_并发编程

---

## 速记

- happens-before 原则和 as-if-serial 语义是JMM中的重要概念，它们都与程序执行的顺序性和一致性有关，但它们的焦点和适用范围有所不同。

### Happens-before 原则

- happens-before 原则是 Java 内存模型（JMM）的一部分，用于确定多线程环境中操作之间的顺序关系。
- **happens-before原则主要用于多线程环境，确保线程间的内存可见性和有序性。

### As-if-serial 语义

- As-if-serial 语义是一种保证单线程程序的行为就像代码按顺序执行一样的概念。
- **as-if-serial语义保证了单线程中，指令重排是有一定的限制的，而只要编译器和处理器都遵守了这个语义，那么就可以认为单线程程序是按照顺序执行的。
- 所以由于synchronized修饰的代码，同一时间只能被同一线程访问。那么也就是单线程执行的。所以，可以保证其有序性。
