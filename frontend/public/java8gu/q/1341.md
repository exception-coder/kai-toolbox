# ✅Redis Cluster 中使用事务和 lua 有什么限制？

> 题号：1341 ｜ 分类：06_Redis

---

## 速记

- Redis Cluster采用主从复制模式来提高可用性。
- 这个思想就和我们在MySQL 中做分库分表是一样的，都是通过一定的分片算法，把数据分散到不同的节点上进行存储。
- **那么和 MySQL 对跨库事务支持存在限制一样，在 Redis Cluster 中使用事务和 Lua 脚本时，也是有一定的限制的。**
- **在 Redis Cluster 中，事务不能跨多个节点执行**。
- 和事务相同，**执行 Lua 脚本时，脚本中访问的所有键也必须位于同一节点。
- 当我们要跨节点执行 lua 的时候，会报错提示：command keys must in same slot。
